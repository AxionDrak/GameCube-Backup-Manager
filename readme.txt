
Sobre
=====

O GameCube Backup Manager permite gerenciar arquivos de jogos em ISO e GCM 
para o console Nintendo GameCube e transferir os arquivos para o formato 
correto suportado pelo programa Nintendont.

Também é possível baixar e transferir as capas dos respectivos jogos gerenciados 
pelo GameCube Backup Manager, além de muitas outras funções disponíveis.

O GameCube Backup Manager é construído com base no conhecimento de outros que abriram o caminho, código compartilhado e conhecimento.
