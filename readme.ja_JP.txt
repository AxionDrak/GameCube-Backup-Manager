
概要
=====

GameCube Backup Managerを使用すると、ISO及びGCMゲームファイルを管理できます。
ゲームキューブに移動し、ファイルを正しい形式に転送します
Nintendont programによってサポートされています。

それぞれのスキンをダウンロードして転送することも可能です
他の多くのゲームに加えて、GameCube Backup Managerによって管理されるゲーム
利用可能な機能。

GameCube Backup Managerは、道を切り開き、コードと知識を共有した他の人々の知識に基づいて構築されています。
